역학적 시스템 : 자연에 존재하는 여러 가지 힘이 물체들 사이에 상호작용하면서 일정한 운동 체제를 유지하는 시스템

중력 : 질량이 있는 모든 물체 사이에 서로 당기는 힘
부력 : 액체, 기체, 유체 속에 있는 물체를 연직 위로 뜨게 하는 힘
전자기력 : 전하를 띤 물체 사이에 서로 밀거나 당기는 힘
자기력 : 자석이나 자성체 사이에 서로 밀거나 당기는 힘
마찰력 : 바닥면과 물체의 상호 작용으로, 물체의 운동을 방해하는 힘
탄성력 : 탄성을 가진 물체가 변형되었을 때 원래 상태로 되돌아가려는 힘
저항(공기저항) : 공기중에서 물체가 운동할 때 물체의 운동을 방해하는 힘

이동거리와 변위
    이동거리 : 물체가 실제로 이동한 거리
    변위 : 위치의 변화, 처음 위치와 나중 위치의 직선 거리와 방향

속력과 속도
    속력 : 물체가 단위시간 동안 이동한 거리, 걸린시간/이동거리  = t, m/s
    속도 : 물체가 단위시간 동안의 위치 변화, 시각의 변화량/변위, m/s
    거리 = 속력 * 시간
    변위 = 속도 * 시간

평균속력 : 물체가 운동하는 도중의 속력 변화는 고려하지 않고 전체 이동 거리를 걸린 시간으로 나누어 구한 속력
    = 걸린시간/전체이동거리 = t/s, 단위 : m/s
    순간속력 : 특정 시각에서의 물체의 속력

가속도 : 물체의 단위 시간 동안의 속도 변화량
    가속도 * 걸린시간 = at
    속도 증가 -> 속도 변화량 : + -> 가속도 : a > 0
    속도 감소 -> 속도 변화량 : - -> 가속도 : a < 0
    속도 일정 -> 속도 변화량 : 0 -> 가속도 : a = 0

등속 직선 운동 : 일직선상에서 같은 속도으로 움직이는 운동 가속도가 0인 운동, 순간속력과 평균속력이 항상 일정한 운동(예: 에스컬레이터)

이동거리-시간 그래프 : 이동거리가 시간에 비례, 직선의 기울기 = 속력 => 속력이 일정, (왼쪽 아래부터 오른쪽 위로 가는 대각선임) 시간/거리
속력-시간 그래프 : 속력이 일정, 넓이 = 거리, 이동거리는 시간에 비례, 기울기 = 가속도 <0 

등가속도 직선 운동 : 일직선상에서 일정한 가속도으로 움직이는 운동, 속도변화량이 항상 일정한 운동, 속력이 일정하게 증가 혹은 감소하는 운동
이동거리-시간 그래프 : 이동거리가 점점증가 혹은 감소한다. 기울기 = 속력(시간/거리)
속력-시간 그래프 : 속력이 일정하게 증가 혹은 감소한다. 넓이 = 이동거리, 기울기 = 가속도 = a
가속도-시간 그래프 : 가속도는 일정하다. 넓이 = 속도 변화량, 속도변화량 : +, 속도변화량 : -

힘 : 물체의 모형이나 운동상태를 변화시키는 원인, 단위 : 뉴턴
힘의 크기 : 화살표 길이, 방향 : 화살표가 가리키는 방향 앞뒤는 +,-를 이용, 힘의 작용점 : 화살표의 시작점으로 표현
알짜힘(합력) : 물체에 작용하는 여러 힘을 마치 하나의 힘으로 보기 위해 모두 합친 것
힘과 가속도의 관계(가속도의 법칙) : 시그마F = M*a

관성 : 물체가 원래의 운동 상태를 유지하려고 하는 성질 -> 정지해 있던 물체는 계속 정지하려고 하고, 운동하는 물체는 계속 운동하려는 성질
벡터 + 벡터(대각선)
힘과 운동의 관계 : 
    등속 직선 운동은 가속도가 0인 운동으로, 물체에 작용하는 힘이 0이다.
    등가속도 직선 운동은 가속도의 크기가 일정한 운동을 나타내는데, 이는 일정한 힘이 계속 작용하고 있음을 의미.

만유인력 : 질량을 가진 두 물체 사이에 서로 잡아 당기는 힘, F = G * r^2/m1 * m2
특징 : 
    물체가 서로 접촉하거나 떨어져 있어도 작용한다.
    지구 표면에서의 중력의 방향은 지구 중심(연직) 방향이다.
    천체의 질량과 크기에 따라 중력의 크기가 다르다.
    두 물체 사이의 거리가 가까울수록 만유인력의 크기는 커진다.
중력을 받는 물체의 운동
    물체에 힘이 작용한다 : 물체에 중력이 연직으로 작용한다.
    힘은 가속도를 만들어낸다 : 지구 내의 물체는 연직 아래 방향으로 중력가속도(g)를 받는다.
    가속도가 결정되면 운동이 결정된다 : 가속도가 중력가속도로 일정한 등가속도 직선운동을 한다.
    운동의 속도-시간 그래프를 통해 시간, 위치, 속도, 가속도의 정보를 찾는다 : 기울기가 중력가속도인 속도-시간 그래프를 그려 위치(면적), 속도(y좌표), 가속도(기울기)를 알아낸다.

자유낙하 : 공기저항을 무시할 때 물체가 중력만 받아 낙하하는 운동, 연직 방향의 일정한 중력이 계속 작용하는 운동, 일정한 가속도를 가지고, 속도가 점점 증가하는 등가속도 직선 운동
자유낙하 해석 : 시간에 따라 공의 이동거리가 점점 증가하면서 속도가 일정한 비율(중력가속도)로 빨라진다.
    가속도 : 기울기 = 중력가속도 = 9.8m/s^2
    속력 : gt -> 시간에 따라 일정하게 증가
    이동거리 : t*9.8 * 2/1 = 2/1gt^2 -> 시간에 따라 점점 증가
공기저항을 무시할 때, 운동방향과 중력의 방향이 나란하지 않으므로 포물선 퀘적을 그리며 운동한다.
수평 방향으로 힘이 작용
연직 방향으로만 힘이 작용
수평 방향은 등속 직선 운동, 
연직 방향은 등가속도 직선 운동을 한다.
두 운동을 따로 해석하되, 시간을 매개로 연결지어 줄 수 있다.
연직방향(등가속도) : 가속도 : 기울기(g), 속력 : gt, 이동 거리 : 2/1gt^2
수평운동(등속운동) : 가속도 : 0, 속력 : v-v0, 이동거리 : vt

결론 :
    자유 낙하 하는 물체는 등가속도 운동을 한다.
    수평 방향으로 던진 물제는
        수평방향으로는 등속 운동을 하고,
        연직방향으로는 등가속도 운동을 한다.
